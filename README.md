# Ejercicios TypeScript

## Descripción

Este proyecto contiene una colección de ejercicios prácticos para aprender y practicar TypeScript. Los ejercicios están organizados por niveles de dificultad: básico, intermedio y avanzado.

## Requisitos Previos

- [Node.js](https://nodejs.org/) (versión 14 o superior)
- [TypeScript](https://www.typescriptlang.org/) (instalado globalmente)

## Instalación

1. Clona este repositorio:
   ```bash
   git clone https://github.com/CarlosSalazar34/ejercicios-typescript.git
   cd ejercicios-typescript
   ```

2. Instala TypeScript globalmente (si no lo tienes):
   ```bash
   npm install -g typescript
   ```

## Ejecución

Para compilar y ejecutar el archivo principal:

```bash
tsc main.ts
node main.js
```

Opcionalmente, puedes usar `ts-node` para ejecutar directamente sin compilar:

```bash
npm install -g ts-node
ts-node main.ts
```

## Estructura de Ejercicios

### Nivel Básico
- Variables y tipos de datos
- Funciones con parámetros opcionales y valores por defecto
- Arrays y objetos

### Nivel Intermedio
- Interfaces para definir contratos de objetos
- Clases con constructores y herencia
- Funciones genéricas

### Nivel Avanzado
- Módulos para organizar el código
- Programación asíncrona con async/await
- Decoradores para metaprogramación
- Tipos avanzados y condicionales
